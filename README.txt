
PHP MEDIA GALLERY (DROP-IN)

1. Upload all files to your server root or a subfolder.
2. Place JPG/PNG/MP4 files inside the /media folder.
3. Ensure PHP GD extension is enabled.
4. Protect with .htaccess + .htpasswd (recommended).
5. Thumbnails auto-generate on first load.

Features:
- Pagination
- Sorting
- Filename search
- Date-based grouping
- Mobile friendly
